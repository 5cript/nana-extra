\subsection{Detailed Description}
Draw a 2\-D plot. 

The plot contains one or more traces.

Each trace can be of one of three types\-:


\begin{DoxyItemize}
\item Plot\-: succesive y-\/values with line drawn between them.
\item Scatter\-: succesive x,y-\/values with box around each point
\item Realtime\-: a specified number of the most recent y-\/values
\end{DoxyItemize}

Any number of plot and scatter traces can be shown together, only one realtime trace may be present in a plot.
